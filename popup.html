Frequently Asked Questions

Different hotel booking websites label amenities and filters in slightly different ways. For instance, Hotels.com might list a general option like “Outdoor space,” while Booking.com separates it into more specific options like “Balcony” and “Garden.”

To make your filters work reliably, the Smart Filter extension includes multiple versions of similar options. That way, you can cover all possible variations.

✅ Tip: Just select the options that best match your intent. The Smart Filter extension will automatically apply the right equivalent on each site.

For example, if you’re looking for a garden, select both “Outdoor space” and “Outdoor space – Garden” to ensure it works across all supported platforms.

To keep things simple and user-friendly, the extension focuses on the most commonly used filters across booking platforms. Some less common or niche options are omitted to avoid overwhelming users.

If you think a particular filter is important, feel free to reach out! If more users request the same filter, it will likely be added in a future update.
